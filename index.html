Volca is a "Source-First" engine. You own the code.

1. Clone / Copy src/volca.ts & src/shaders.wgsl
2. Drop them into your Vite/React/Next.js folder.
3. Initialize:
import { VolcaCore } from './volca';

// Use on ANY existing canvas
const engine = new VolcaCore(document.getElementById('bg'));
engine.boot({ count: 500000 });
