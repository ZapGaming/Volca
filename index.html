What is Volca?

Volca is a massively parallel graphics engine built on WebGPU. Unlike Three.js or Babylon, Volca moves 100% of physics calculations (gravity, drag, turbulence) to the GPU Compute Shaders, allowing for cinema-quality explosions and simulations in the browser.

Installation

npm install volca-engine gl-matrix

Initialize the core in your TypeScript project:

import { VolcaCore } from 'volca-engine';

const canvas = document.querySelector('#gpu-view');
const engine = new VolcaCore(canvas);

// Boot with default "Inferno" preset
engine.boot({ preset: 'inferno' });

The ZCSS Standard

Volca introduces Zandy Cascading Style Sheets. Control the physics engine using CSS-like syntax.

simulation.zcss
/* Define the physics container */
#main-simulation {
    /* Rendering Backend */
    renderer: volca-gpu;
    
    /* Physics Properties */
    physics-gravity: -9.8;     /* Earth Gravity */
    physics-turbulence: 5.2;   /* Chaos Factor */
    physics-viscosity: 0.1;    /* Air resistance */
    
    /* Particle Count */
    spawn-count: 2000000;
    
    /* Aesthetics */
    blend-mode: additive;
    color-scheme: cyber-blue;
}

Importing 3D Models (.glb)

Emit particles from 3D geometry vertices dynamically.

#hero-mesh {
    /* Async Model Loading */
    source: url('/assets/tank_destroyed.glb');
    
    /* Explode the model */
    behavior: mesh-surface-emit;
    velocity-scale: 15.0;
}
